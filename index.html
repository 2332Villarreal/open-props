@import "open-props";

.button {
  padding-inline: var(--size-5);
  padding-block: var(--size-3);

  color: var(--blue-0);
  background-color: var(--blue-7);
  border-radius: var(--border-radius-round);
  box-shadow: var(--shadow-1);
  
  @media (--motionOK) {
    transition: background .3s var(--easinout-3);
  }

  @media (--OSdark) {
    background: var(--blue-1);
    color: var(--blue-9);
  }
}

Everything's optional
Grab some of the props, or all of em.
No magic units
Design consistency without thinking
Easy to customize
Extend, overwrite, adapt, etc.. no prob.
Easy to remember
Try to keep the API surface area small and legible

Usage Sample

button.blue {
  color: var(--blue-6);
  background-color: var(--blue-0);
  border: 1px solid var(--blue-1);
  text-shadow: 0 1px 0 var(--blue-2);

  &:hover {
    background-color: var(--blue-1);
  }
}
