Grab n' go
CSS custom properties.

Easy to remember, custom property utility values.


@import "open-props";

.button {
  font-size: var(--font-size-fluid-1);

  padding-inline: var(--size-5);
  padding-block: var(--size-3);

  color: var(--blue-0);
  background: var(--blue-7);
  border-radius: var(--border-radius-round);
  box-shadow: var(--shadow-3);
  
  @media (--motionOK) {
    transition: background .3s var(--easinout-3);
  }

  @media (--OSdark) {
    background: var(--blue-1);
    color: var(--blue-9);
  }
}

Getting Started

PostCSS and pre-built version are available.

Installation

Grab the props from a URL or NPM

NPM
$ npm install open-props

Import pre-built bundle

@import "open-props";

Import PostCSS unbundled

@import "open-props/src/index.css";
CDN

Import pre-built bundle


@import "https://unpkg.com/open-props";

or


<link rel="stylesheet" href="https://unpkg.com/open-props"/>

Naming Convention

Usually you'll just be adjusting between 1 and 6, cycling through to find the right value.

  1. Colors range from 0 (nearly white) to 9 (nearly black)
  2. var(--size-1) Sizes between 1-n are "safe" sizes
  3. var(--size-00) for allowed negative values
  4. var(--font-light) and `@media (--portrait)`

Colors

Props includes Open Color (Version 1.9.1): an open-source color scheme. Optimized for UI like font, background, border, etc.


.open-color-pallete {
  color: var(--blue-6);
  background-color: var(--blue-0);
  border: 1px solid var(--blue-1);
  text-shadow: 0 1px 0 var(--blue-2);
}
