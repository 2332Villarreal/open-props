  }
}

Colors

Open Props includes Open Color (Version 1.9.1): an open-source color scheme. Optimized for UI.

Open devtools, open the color picker, set the swatches to the page's custom properties, and enjoy picking from the set!
Vars Sample

.pinks {
  --pink-0: #fff0f6;
  ...
  --pink-9: #a61e4d;
}
            

.oranges {
  --orange-0: #fff4e6
  ...
  --orange-9: #d9480f;
}
Usage Sample

button.blue {
  color: var(--blue-6);
  background-color: var(--blue-0);
  border: 1px solid var(--blue-1);
  text-shadow: 0 1px 0 var(--blue-2);

  &:hover {
    background-color: var(--blue-1);
  }
}

Easing

5 strengths for the classic ease, easeout, easein, and easeinout. Some extra easings too, a space for mock elasticity and physics.

Ease

--ease-1: cubic-bezier(.25, 0, .5, 1);
--ease-2: cubic-bezier(.25, 0, .4, 1);
--ease-3: cubic-bezier(.25, 0, .3, 1);
--ease-4: cubic-bezier(.25, 0, .2, 1);
--ease-5: cubic-bezier(.25, 0, .1, 1);
Usage Sample

.slight-ease {
  animation: fade-in 300ms var(--ease-1);
}

.dramatic-ease {
  animation: fade-in 1s var(--ease-5);
}
